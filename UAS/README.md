a. Judul Proyek

Mini Game 3D – Subway Surfers (Transformasi Geometri 3D)

b. Konsep Grafika yang Digunakan

Mini project ini menerapkan grafika komputer 3D berbasis transformasi geometris, meliputi:

1. Translasi 3D
   Digunakan untuk menggerakkan karakter dan objek di sepanjang lintasan (sumbu X, Y, dan Z), seperti gerakan maju, ke kiri, dan ke kanan layaknya permainan Subway Surfers.

2. Rotasi 3D
   Digunakan untuk efek animasi, seperti putaran karakter atau rintangan, dengan rotasi terhadap sumbu X, Y, atau Z.

3. Skala 3D
   Digunakan untuk memperbesar atau memperkecil objek, misalnya efek power-up atau perubahan ukuran karakter dan lingkungan.

4. Refleksi 3D
   Digunakan untuk mencerminkan objek terhadap sumbu atau bidang tertentu, misalnya efek mirror pada lingkungan atau objek tertentu.

Seluruh transformasi dilakukan secara matematis menggunakan koordinat dan matriks transformasi 3D, sehingga bersifat edukatif dan sesuai konsep dasar grafika komputer.

c. Cara Menjalankan Program

1. Pastikan Python sudah terinstal di komputer.
   
2. Install library yang dibutuhkan (misalnya PyOpenGL, GLUT, atau library grafika yang digunakan):
   pip install PyOpenGL PyOpenGL_accelerate
   
3. Buka folder project dan jalankan file utama :
   python Uas3D.py
   
4. Jendela aplikasi 3D akan muncul dan mini game dapat dijalankan.
